# pypy-lang-scheme

fork from [lang-scheme](https://bitbucket.org/pypy/lang-scheme)

## Quick Start

Build. If you have pypy, you can use it.

```console
$ python rpython/bin/rpython targetscheme.py
```

and Run

```console
$ ./scheme-c sample.ss
<<W_Boolean object at 0x7f61f0e7a020>>
<<W_Boolean object at 0x7f61f0e7a020>>
```
